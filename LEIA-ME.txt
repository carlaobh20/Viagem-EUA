FATIA 4 - Corrige o card dos EUA piscando no login
====================================================

O QUE MUDA: so 1 arquivo -> components/views/Viagens.js

O QUE FOI FEITO:
- A foto do card piscava porque era rebuscada quando o total de
  gastos chegava. Agora a foto e pre-carregada no cache do navegador
  assim que a lista de viagens carrega, entao nao pisca mais.
- So foi ADICIONADO um pre-carregamento. Nada do resto mudou
  (convite, apagar, trocar foto, total - tudo igual).

COMO INSTALAR:
1. Descompacte
2. Copie a pasta "components" por cima da sua pasta
3. No PowerShell:
     cd "C:\MEUS PROJETOS\Viagem-EUA"
     npm run dev
4. Abra localhost:3000, faca login, e veja se o card dos EUA
   ainda pisca. Pode precisar de Ctrl+Shift+R pra limpar o cache
   antigo do PWA na 1a vez.
5. Se parou de piscar e nada quebrou, push:
     git add .
     git commit -m "V5.3 - corrige flicker do card no login"
     git push

OBS: na PRIMEIRA vez que abrir, a foto ainda pode dar uma carregada
(porque o cache esta vazio). A partir da 2a vez, nao pisca mais.
