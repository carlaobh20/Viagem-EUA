FATIA 7 - Ajustes na tela Viagens
==================================

O QUE MUDA: so 1 arquivo -> components/views/Viagens.js

O QUE FOI FEITO:
1. Tirei o aviaozinho do lado de "Ola, Carlos!"
2. Tirei o botao "Criar nova viagem" de baixo. Agora so existe o
   botao + no canto superior direito - deixei ele maior e mais
   bonito (borda branca sutil, sombra melhor).
3. A secao de historico agora se chama "VIAGENS REALIZADAS"
   (antes era "PASSADAS"). Ela aparece sozinha quando voce tiver
   alguma viagem com a data de volta ja passada.

COMO INSTALAR:
1. Descompacte
2. Copie a pasta "components" por cima da sua pasta
3. cd "C:\MEUS PROJETOS\Viagem-EUA"
4. npm run dev
5. Abra localhost:3000. A tela que abre e a de Viagens. Veja:
   - "Ola, Carlos!" sem aviao
   - so o botao + no canto (sem o "Criar nova viagem" embaixo)
   - clique no + pra confirmar que ainda cria viagem
6. Se ficou bom:
     git add .
     git commit -m "V5.6 - ajustes na tela Viagens (+ no canto, historico)"
     git push

OBS: a secao "VIAGENS REALIZADAS" so vai aparecer quando existir uma
viagem ja concluida. Como a Viagem EUA ainda nao aconteceu, e normal
ela nao aparecer agora.
