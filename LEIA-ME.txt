COMO INSTALAR:
1. Descompacte este zip
2. Copie a pasta "components" por cima da sua pasta do projeto
   (so o Viagens.js vai ser substituido, mais nada e tocado)
3. No PowerShell:
     cd "C:\MEUS PROJETOS\Viagem-EUA"
     npm run dev
4. Abra localhost:3000 (tela que abre primeiro). Veja:
   - aparece seu nome no "Ola, ___!" ?
   - da pra ler bem o texto sobre a foto agora?
5. Se subir sem erro vermelho e estiver bom, faca o push:
     git add .
     git commit -m "V5.2 - nome no Ola e leitura do header Viagens"
     git push

NAO use git push --force em hipotese nenhuma.
